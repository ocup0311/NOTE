# [快捷鍵 - 展開＆縮起程式碼](https://stackoverflow.com/questions/42660670/collapse-all-methods-in-visual-studio-code)

- `Ctrl` + `K` + `0`: fold all levels (namespace, class, method, and block)
- `Ctrl` + `K` + `1`: namspace
- `Ctrl` + `K` + `2`: class
- `Ctrl` + `K` + `3`: methods
- `Ctrl` + `K` + `4`: blocks
- `Ctrl` + `K` + `[` or `Ctrl` + `k` + `]`: current cursor block
- `Ctrl` + `K` + `j`: UnFold
