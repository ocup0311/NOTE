## 1. Web

- TCP： 辨認 port，例如 80 - HTTP、443 - HTTPS

- Http-Server：處理將 TCP/IP 與 HTTP 結合使用 (57:00)

- Ephemeral port (臨時埠) (1024 ～ 65535)

  - client 向 server 發請求，進行 TCP 三方交握時，OS 會自動分配臨時 port
  - server 接下來與 client 溝通，就會傳回到 client 的那個 port
